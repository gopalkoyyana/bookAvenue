# BookPartyHall 🏰✨

**BookPartyHall** is a premium, modern venue booking platform built with specific focus on bringing global reach with local precision. It simplifies the process of finding and booking the perfect venue for weddings, parties, corporate events, and more.

## 🌟 Key Highlights

### 1. Global Reach with Local Precision 🌍
- **Comprehensive Coverage**: detailed listings for **India** (Andhra Pradesh & Telangana) and support for locations in **USA, UK, Canada, Australia, and Europe**.
- **Smart Autocomplete**: Context-aware search that dynamically filters States and Cities based on country selection.

### 2. Stunning "Premium" Aesthetics 💎
- **Glassmorphism UI**: A visually rich interface with `backdrop-blur` effects, smooth animations, and a high-quality luxury feel.
- **Modern Tech Stack**: Built with **Next.js 16**, **Tailwind CSS 4**, and **Lucide Icons** for a fully responsive and accessible experience.

### 3. Real-Time Availability Engine 📅
- **Instant Status**: Check venue availability instantly for your specific event date.
- **Visual Feedback**: 
    - 🟢 **Green Pulse**: Available
    - 🔴 **Red**: Booked
    - Interactive indicators on both venue cards and map markers.

### 4. Interactive Map-First Search 🗺️
- **Dynamic Mapping**: Powered by **Leaflet**, allowing users to visualize venues within a customizable search radius (up to 50km).
- **Flexible Views**: Switch seamlessly between **Split View**, **Map Only**, and **List Only** modes.

### 5. Secure Venue Listing Portal 🔐
- **Easy Onboarding**: Dedicated flow for venue owners to list their properties.
- **Admin Security**: Protected by a secure Admin Access Code system to ensure listing quality.

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Technology Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Maps**: React Leaflet / Leaflet
- **Language**: TypeScript
